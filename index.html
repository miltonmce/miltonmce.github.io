Sobre mí

Me interesa crear soluciones que realmente funcionen. Prefiero lo simple pero bien hecho. Me enfoco en entender el contexto, proponer caminos viables y construir desde lo que ya existe, sin complicar lo innecesario. Trabajo desde la lógica y la claridad, combinando lo técnico con lo práctico.

Experiencia

Desde 2016 he trabajado como desarrollador Python freelance, creando herramientas para resolver problemas reales: desde migrar sistemas antiguos escritos en VB y PHP, hasta construir aplicaciones modernas con Python y bases de datos como MySQL y PostgreSQL.

Recientemente, he tenido la oportunidad de colaborar con Asudir037, desarrollando soluciones para la gestión de sistemas hídricos. Ahí participé en la construcción de herramientas para controlar volúmenes de agua, gestionar pozos, generar reportes técnicos y procesar autorizaciones de riego. También integré el uso de brokers de mensajería para recolectar y almacenar datos enviados desde PLCs (controladores lógicos programables), facilitando el monitoreo en tiempo real.

También lideré proyectos en Corcab, donde dirigí el desarrollo de plataformas empresariales usando Python, web2py, APIs REST y microservicios. Participé en la implementación de facturación electrónica con Timbox y enfoqué mi trabajo en hacer que los sistemas fueran sostenibles a largo plazo.

Además de programar, he colaborado en cursos, comunidades y trabajos de investigación. Contribuí al Arctic Code Vault con un proyecto de lectura/escritura de tarjetas MSR605, participé en la creación de un curso sobre web2py para Latinux y Semnix, e incluso formé parte de una investigación doctoral sobre integridad de datos.

Habilidades

  • Python (desde 2.7 hasta 3.x) — Desarrollo web, automatización, procesamiento de datos, generación de reportes PDF y gráficos.
  • Frameworks y arquitecturas — web2py, Django REST Framework, microservicios, arquitectura orientada a dominios (DDD), APIs RESTful.
  • Frontend — jQuery, HTMX, Vue.js (integración progresiva), uso de Bootstrap y Tailwind CSS, manejo de componentes reutilizables.
  • Bases de datos — Diseño y consultas avanzadas con PostgreSQL y MySQL. Conexión eficiente entre backend y modelos complejos.
  • Testing — Pruebas automatizadas en Django, pruebas de integración y validación de lógica de negocio compleja.
  • Mensajería y automatización — Uso de brokers (como MQTT o similares) para recibir información de PLCs, almacenamiento y monitoreo en tiempo real.
  • Sistemas hídricos — Modelado de datos para pozos, volúmenes de agua, arrendamientos, subarrendamientos, autorizaciones de riego, transferencia temporal de volúmenes y lectura mensual de medidores.
  • Generación de reportes — Uso de PyFPDF para PDFs personalizados y PyGal para visualización de datos con gráficos limpios y exportables.
  • Control de versiones y colaboración — Git y GitHub. Contribuciones al Arctic Code Vault.
  • Documentación y levantamiento de requerimientos — Experiencia directa con usuarios para transformar procesos reales en soluciones funcionales.
